# Coding Boilerplate

## Project setup

```
$ git clone git@github.com:tandems/coding_boilerplate.git
$ cd coding_boilerplate
```

### Initialize the Project
```
$ make init
$ make up
```

Access to http://localhost:8080

Enjoy your development :)

### Lint files

```
$ make lint
```

### Format files

```
$ make format
```
